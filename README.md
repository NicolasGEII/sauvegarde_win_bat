# sauvegarde_win_bat
Script bat pour sauvegarder un dossier sous format zip avec horodatage vers un dossier source (Drive, NAS, autres espace de stockage), avec log du script.

## Execution du script
1. Modifier les champs SOURCE et DEST par le chemin de source du dossier (ex: "C:\Users\you\Document\fichier_important") et par la destination des sauvegardes (ex: "Z:\Sauvegardes\fichier_important").
2. Avec le planificateur de tâches de windows planifier une tâche en éxécutant ce script une fois par (jour/semaine/mois) à 17h00 pour sauvegardé et archivé le dossier.
